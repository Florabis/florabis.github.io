# Welcome
This repository holds the website for Florabis a state in the Imperial Federation. You can access the website [here](https://florabis.github.io/).

## Contributing
`Note: You can only contribute if you are permitted. To request permission, go bother: borito185`
You can contribute either by editing the repository directly or, 
for an easier alternative, use the Content Management Service at [/admin/index.html](https://florabis.github.io/admin/index.html)

Currently, you can only sign in using a token. 
But in the near future that will change once GitHub adds [this feature](https://github.com/github/roadmap/issues/1153).

## Organization
This section explains how the repository is organized.

### Tech Stack
The website use 3 main tools:
1. VitePress - A framework that generates a website based of MarkDown language files.
2. GitHub Pages - GitHub's built-in and free hosting platform
3. Sveltia CMS - A Content Management Service to make editing the website easier for non-technical people.
